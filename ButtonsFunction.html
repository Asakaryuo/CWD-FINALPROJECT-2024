BUTTONS AND FUNCTIONS

Buttons and functions at TFT are possible since we are using an Arduino module to create operations that we can easily manipulate to create a whole and functioning button
Adafruit_GFX
Adafruit_GFX_Button Create Button Call out Names
Adafruit_GFX_Button ON
TouchScreen ts
TouchScreen(XP,YP,XM,YM,300)
This lets you initialize the Touchscreen library
Button Init
Button Callout name in Adafruit_GFX.initButton(&tft,X pos,Y pos,Width,Height,Outline Color,Fill Color,Text Color, Text printed, Textsize);
ON.initButton(&tft, 60, 200, 100, 40, WHITE, CYAN, BLACK, "ON", 2);
To be able to make this buttons appear whether it is active or not put this inside the void setup()
Format Name of the Button Callout.drawButton(true/false);
Sample ON.drawButton(true); or ON.drawButton(false);
The purpose of a true and a false specification will show the button whereas if it is called in false state it would act as an inactive button while in true state shows it as if it is an active or pressed button. Let's Try it putting these codes into work. 
        
//BUTTONS

//Libraries start with #include 
#include <'Adafruit_GFX.h'>
#include <'MCUFRIEND_kbv.h'>
#include <'TouchScreen.h'>
            
//Use "MCUFRIEND_kbv (name); create a name that you will use in the call out of the code later on
MCUFRIEND_kbv tft;

//Define the Minimum and Maximum resistance for the Touchscreen threshold
#define MINPRESSURE 100
#define MAXPRESSURE 1000


//This is the list of the code if u want to add more just use the #define function with a format of #define-"COLOR NAME"-Hex Value
#define BLACK       0x0000      /*   0,   0,   0 */
#define NAVY        0x000F      /*   0,   0, 128 */
#define DARKGREEN   0x03E0      /*   0, 128,   0 */
#define DARKCYAN    0x03EF      /*   0, 128, 128 */
#define MAROON      0x7800      /* 128,   0,   0 */
#define PURPLE      0x780F      /* 128,   0, 128 */
#define OLIVE       0x7BE0      /* 128, 128,   0 */
#define LIGHTGREY   0xC618      /* 192, 192, 192 */
#define DARKGREY    0x7BEF      /* 128, 128, 128 */
#define BLUE        0x001F      /*   0,   0, 255 */
#define GREEN       0x07E0      /*   0, 255,   0 */
#define CYAN        0x07FF      /*   0, 255, 255 */
#define RED         0xF800      /* 255,   0,   0 */
#define MAGENTA     0xF81F      /* 255,   0, 255 */
#define YELLOW      0xFFE0      /* 255, 255,   0 */
#define WHITE       0xFFFF      /* 255, 255, 255 */
#define ORANGE      0xFD20      /* 255, 165,   0 */
#define GREENYELLOW 0xAFE5      /* 173, 255,  47 */
#define PINK        0xF81F

//This is for the Call out Pins on the Arduino Pins
const int XP = 6, XM = A2, YP = A1, YM = 7;

//This is for the sensitivity of the Touchscreen Module
//NOTE!!! ~~ This differs on TFT Modules check it with the callibration
const int TS_LEFT =907, TS_RT = 136, TS_TOP = 942, TS_BOT = 139;

//Touch Screen Pins in the Arduino
TouchScreen ts = TouchScreen(XP, YP, XM, YM, 300);

//Listing and naming of Buttons
Adafruit_GFX_Button ON,OFF;

//This part focuses on setting up the Touch attributes for the Touchscreen function to work
int pixel_x, pixel_y;  
bool Touch_getXY(void)
{
TSPoint p = ts.getPoint();
pinMode(YP, OUTPUT);      //restore shared pins
pinMode(XM, OUTPUT);
digitalWrite(YP, HIGH);   //because TFT control pins
digitalWrite(XM, HIGH);
bool pressed = (p.z > MINPRESSURE && p.z < MAXPRESSURE);
if (pressed) {
pixel_x = map(p.x, TS_LEFT, TS_RT, 0, tft.width()); //.kbv makes sense to me
pixel_y = map(p.y, TS_TOP, TS_BOT, 0, tft.height());
}
return pressed;
}

void setup() {
// put your setup code here, to run once:
uint16_t ID = tft.readID(); //This is used to read the driver chips inside the TFT for it to run smoothly
if (ID == 0xD3D3) ID = 0x9486; //
tft.begin(ID);    //Use this to Initialize the TFT
tft.fillScreen(BLACK);  //This is normally used to restart the whole Screen

//Button Inits
            
ON.initButton(&tft,  60, 200, 100, 40, WHITE, CYAN, BLACK, "ON", 2);
OFF.initButton(&tft, 180, 200, 100, 40, WHITE, CYAN, BLACK, "OFF", 2);


ON.drawButton(false); //drawButton functions
OFF.drawButton(false); //drawButton functions
}

void loop(){
bool down = Touch_getXY();
ON.press(down && ON.contains(pixel_x, pixel_y));
OFF.press(down && OFF.contains(pixel_x, pixel_y));

if (ON.justReleased())
ON.drawButton();
if (OFF.justReleased())
OFF.drawButton();
if (ON.justPressed()) {
ON.drawButton(true);
tft.fillRect(40, 80, 160, 80, GREEN);
}
if (OFF.justPressed()) {
OFF.drawButton(true);
tft.fillRect(40, 80, 160, 80, RED);
}
}
This is how your TFT should look like!
The color above the buttons should differ every time you touch a different button Green with ON while Red with OFF.
