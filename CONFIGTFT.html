TFT Configuration and Specification

TFT Display differ when it comes to their sizes such as 3.5 inch TFT Display which are best suited for an Arduino AtMega 2560. Since you can just mount it as a TFT Arduino Shield it is much easier and doesnt need intricate wirings to make the TFT Display work.

TFT Screen Orientation



In the Arduino IDE you can set the desired orientation whether your screen should be on Portrait or Landscape by using the .setRotation() function

Code Orientation
setRotation(0) Portrait
setRotation(1) Landscape
setRotation(2) Inversed Portrait
setRotation(3) Inversed Landscape
