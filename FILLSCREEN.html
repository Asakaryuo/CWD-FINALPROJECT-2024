FILLSCREEN TFT FUNCTION


Fillscreens and shapes in TFT Display are used to reset the whole screen or just to make a colorful display. Unlike the text this requires a longer format of code to use.
FillRect Code
tft.FillRect(X pos,Y pos,Width,Height,Color)
tft.FillRect(10,20,300,50,RED)


FillScreen Code
tft.FillScreen(Color you want to Fill the Screen With)
tft.FillScreen(BLACK)
By using the the codes above, just like in the Texts and Colors, we are going to use the #define function which we are going to use for the defining of colors to be used for the FillScreen and FillRect functions. 
        
//RECTANGLES & FILLSCREEN

//Libraries start with #include <'Library.h'>
#include <'Adafruit_GFX.h'>
#include <'MCUFRIEND_kbv.h'>
#include <'TouchScreen.h'>
            
//Use "MCUFRIEND_kbv (name); create a name that you will use in the call out of the code later on
MCUFRIEND_kbv tft;

//Define the Minimum and Maximum resistance for the Touchscreen threshold
#define MINPRESSURE 100
#define MAXPRESSURE 1000


//This is the list of the code if u want to add more just use the #define function with a format of #define-"COLOR NAME"-Hex Value
#define BLACK       0x0000      /*   0,   0,   0 */
#define NAVY        0x000F      /*   0,   0, 128 */
#define DARKGREEN   0x03E0      /*   0, 128,   0 */
#define DARKCYAN    0x03EF      /*   0, 128, 128 */
#define MAROON      0x7800      /* 128,   0,   0 */
#define PURPLE      0x780F      /* 128,   0, 128 */
#define OLIVE       0x7BE0      /* 128, 128,   0 */
#define LIGHTGREY   0xC618      /* 192, 192, 192 */
#define DARKGREY    0x7BEF      /* 128, 128, 128 */
#define BLUE        0x001F      /*   0,   0, 255 */
#define GREEN       0x07E0      /*   0, 255,   0 */
#define CYAN        0x07FF      /*   0, 255, 255 */
#define RED         0xF800      /* 255,   0,   0 */
#define MAGENTA     0xF81F      /* 255,   0, 255 */
#define YELLOW      0xFFE0      /* 255, 255,   0 */
#define WHITE       0xFFFF      /* 255, 255, 255 */
#define ORANGE      0xFD20      /* 255, 165,   0 */
#define GREENYELLOW 0xAFE5      /* 173, 255,  47 */
#define PINK        0xF81F

//This is for the Call out Pins on the Arduino Pins
const int XP = 6, XM = A2, YP = A1, YM = 7;
//This is for the sensitivity of the Touchscreen Module
//NOTE!!! ~~ This differs on TFT Modules check it with the callibration
const int TS_LEFT =907, TS_RT = 136, TS_TOP = 942, TS_BOT = 139;

void setup() {
// put your setup code here, to run once:
uint16_t ID = tft.readID(); //This is used to read the driver chips inside the TFT for it to run smoothly
if (ID == 0xD3D3) ID = 0x9486; //
tft.begin(ID);    //Use this to Initialize the TFT
tft.fillScreen(BLACK);  //This is normally used to restart the whole Screen
}
            
            
void loop() {
// put your main code here, to run repeatedly:
            
//tft.fillRect(X-Position,Y-Position,Rectangle Width,Rectangle Height,Rectangle Color)
            
tft.fillRect(10,20,300,50,RED);
            
tft.fillRect(10,80,300,50,BLUE);
            
tft.fillRect(10,140,300,50,YELLOW);
            
tft.fillRect(10,200,300,50,PURPLE);
            
tft.fillRect(10,260,300,50,CYAN);
            
tft.fillRect(10,320,300,50,GREEN);
            
tft.fillRect(10,380,300,50,OLIVE);
            
delay(5000); //Delays are used to delay a command inside the loop where 1000 = 1 second so 5000 here is 5 seconds
            
//fillScreens are used to fill the whole TFT screen of the color of your choice
tft.fillScreen(BLACK);

}
